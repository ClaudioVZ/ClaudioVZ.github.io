Inicio

Software usado en esta página:

Python

Es un lenguaje de programación interpretado cuya filosofía es "código más legible".

Julia

Es un lenguaje de programación de alto nivel y alto desempeño para la computación genérica, técnica y científica.

Project Jupyter

Computación interactiva multilenguaje creado por Fernando Pérez.

GitHub

Es una plataforma de desarrollo colaborativo para alojar proyectos utilizando el sistema de control de versiones Git.

Markdown

Es un lenguaje de marcado ligero creado originalmente por John Gruber y Aaron Swartz.

MathJax

Es una biblioteca javascript que permite visualizar fórmulas matemáticas en navegadores web, utilizando los lenguajes de marcado LaTeX o MathML.

LaTeX

Es un sistema de composición de textos, orientado a la creación de documentos escritos que presenten una alta calidad tipográfica.

Asymptote

Es un lenguaje para crear gráficos vectoriales de alta calidad en formato PostScript, PDF, SVG, o 3D PRC, desarrollado por Andy Hammerlindl, John C. Bowman y Tom Prince.

PGF & TikZ

Es un par de lenguajes para crear gráficos vectoriales mediante una descripción geométrica/algebraica, fue desarrollado por Till Tantau y Christian Feuersänger.

Doconce

Es un lenguaje de marcado creado por Hans Petter Langtangen, es adecuado parar generar texto con fórmulas matemáticas y código fuente.

Pandoc

Es un conversor de documentos de código abierto, fue creado originalmente por John MacFarlane.
